# FlaskAPI-mongodb

To run this app

1. Install requeriments
   ```bash
   pip install pymongo flask_cors flask
   ```
2. Create a envieroment and activate
   ```bash
   python -m venv app_env
   .\app_env\Scripts\activate
   ```
4.  run web server
    ```python
      python app.py
    ```
5. Open the wepage in browser with link: http://127.0.0.1:5000/
